# Visual Asset Generation Prompts

These prompts are for original project-bound assets. The reference screenshots are used only for composition, mood, and UI density. Do not copy any real restaurant logo, real brand mark, real person likeness, or readable third-party signage.

## 1. Background

```text
Use case: photorealistic-natural
Asset type: visual novel background, 16:9 game stage
Primary request: Create an original photorealistic interior background for a Japanese gyudon counter restaurant, inspired by the provided visual novel demo screenshots only in composition and mood.
Scene/backdrop: warm casual beef-bowl restaurant interior at evening, counter on the left, open kitchen behind the counter, menu boards above the counter, small tables and seats on the right, orange fictional shop sign on the back wall.
Style/medium: photorealistic game background, slightly softened cinematic still, realistic lens perspective, high detail.
Composition/framing: wide 16:9 view, camera at seated eye level, enough clean space in the lower third for a dialogue box overlay, no characters, no UI.
Lighting/mood: warm ceiling lights, cozy orange-brown interior, mild depth of field but still readable.
Text: fictional signage only; if text is included, use "GYUDON HOUSE" or abstract unreadable marks. No real brand names.
Constraints: no people, no UI, no dialogue box, no watermarks, no real restaurant logos, no "Yoshinoya", no exact copied signage from references.
Avoid: copyrighted or trademarked logos, real storefront branding, distorted readable menu prices, extra foreground characters.
```

## 2. Character Neutral

```text
Use case: photorealistic-natural
Asset type: visual novel character bust cutout source
Primary request: Create an original fictional Japanese young adult man for a romance visual novel, neutral expression, awkward but gentle personality, not based on any real person.
Subject: slim young adult man with short straight black hair, rectangular black glasses, faint facial hair shadow, small mole-like facial detail, beige-brown plaid button shirt, shoulders and upper torso visible.
Style/medium: photorealistic portrait cutout for visual novel, clean studio-like front lighting, realistic skin and hair, game asset polish.
Composition/framing: centered bust from upper chest to top of head, facing camera, generous padding around the subject.
Lighting/mood: soft neutral lighting, slight shy atmosphere.
Background for removal: perfectly flat solid #00ff00 chroma-key background, no shadows, no gradient, no floor plane.
Constraints: no text, no watermark, no real person likeness, no celebrity, no copied face from the screenshots, no background scene, do not use #00ff00 anywhere in the subject.
Avoid: anime style, caricature, extreme distortion, extra hands, props, logos.
```

## 7. UI Kit

```text
Use case: ui-mockup
Asset type: visual novel UI sprite sheet
Primary request: Create a clean visual novel UI asset sheet inspired by the provided demo screenshots: translucent white dialogue box frame with pink border, black translucent choice button frames, pink name label, dark translucent HUD panels, heart gauge parts, numbered diamond markers, small log/auto/menu icons, and a white clicking hand pointer.
Style/medium: polished 2D game UI assets, semi-transparent glass panels, soft pink highlights, dark rounded buttons, crisp edges.
Composition/framing: sprite sheet layout on a perfectly flat solid #00ff00 chroma-key background, each asset separated with generous padding. No full screen mockup.
Text: avoid readable UI text except simple numerals 1, 2, 3, 4 inside diamond markers if possible.
Constraints: no game screenshot background, no character, no real logos, no watermark, no decorative text, do not use #00ff00 inside the UI elements.
Avoid: cluttered layout, full app mockup, unreadable text blocks, brand names.
```
